About Me

Domain Summaries


  • Domain 1:

    In this Domain we learned basic HTML such as using head, body, and comment tags. When running the code, the browser tells you whether it supports JavaScript or not.

  • Domain 2:

    In this Domain we learned how to customize a webpage. These things include classes, class selectors, IDs, and ID selectors. The 3 different types of CSS are Internal, External, and Inline CSS.

  • Domain 3:

    Domain 3 introduced many tags like h1-6, p, hr, hr, div, span, ul, li,summary, figure, caption, img, a, and footer tags. Footer tags are used to keep information like phone numbers and addresses in order. We also learned hiding photo details, creating alt attributes, and creating forms.

  • Domain 4:

    Domain 4 allowed us to manipulate media such as videos and audio on our page. We learned about the control tag to improve user accessibility and the video tag is used to insert a video.

  • Domain 5:

    Domain 5 taught resizing and positioning using CSS. Height and width is used to adjust the size of an element. Overflow allows the text to go over the div while contain shows the whole thing but in the case of an image, it can be smaller. Auto is the default and adjusts accordingly. Float is used to position an element left or right. Absolute and relative is for layout control.

  • Domain 6:

    In Domain 6, we learned about HTML rulesets. We learned about hierarchy where different tags are used to display different information. We also learned how rulesets are used for efficiency and how rulesets reduce the chance of error with its redundancy.
